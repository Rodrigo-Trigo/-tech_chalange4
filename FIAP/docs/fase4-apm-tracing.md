# APM e Tracing

Este documento explica a instrumentação de microsserviços com OpenTelemetry.

## OpenTelemetry
- Código adaptado para gerar métricas, logs e traces.
- Exemplo: `auth-service/main.go`.

## Integração com APM
- Exportação de dados para Datadog ou New Relic.
- Visualização de Service Map e Traces.

## Evidências
- Prints de traces distribuídos.
- Service Map mostrando dependências entre microsserviços.
